# Contact Management System Automation

## Project Objective
Automate testing of the web application: [Thinking Tester Contact List](https://thinking-tester-contact-list.herokuapp.com/)

## Scope
- Login, Registration, Add/Edit/Delete Contacts
- Session & Navigation testing
- UI & Responsiveness validation
- Input validation & Unicode/emoji support
- Cross-browser testing (Chrome, Firefox, Edge)

## Test Scenarios
1. **Login Functionality** – valid/invalid credentials, empty fields, email format, password masking
2. **Sign Up / Registration** – valid registration, duplicate email, password mismatch, blank fields
3. **Add Contact** – valid input, missing fields, numeric validation, duplicate contacts, form reset
4. **Edit Contact** – edit details, cancel edit, validation
5. **Delete Contact** – delete contact, delete confirmation, persistence after deletion
6. **Session & Navigation** – logout, login state on refresh, back button after logout
7. **UI & Responsiveness** – field alignment, toast/success messages
8. **Input Validation & Security** – max character limits, Unicode and emoji support
